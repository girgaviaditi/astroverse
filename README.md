# Astroverse

Astroverse is a modern Vedic Astrology and Astronomy platform.

## Vision

To build one of the most accurate open-source platforms that combines:

- Vedic Astrology
- Modern Astronomy
- Panchang
- Planetary Motion
- Birth Chart Analysis
- Scientific Sky Information

## Tech Stack

- Next.js
- Node.js
- Express
- PostgreSQL
- Prisma
- Swiss Ephemeris
- TypeScript
